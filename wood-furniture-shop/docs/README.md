# Wood Furniture Shop

Dự án Website Cửa hàng Nội thất Gỗ tĩnh được phát triển hoàn toàn bằng HTML5, CSS3 và JavaScript thuần.

## Tính năng nổi bật
- Giao diện Responsive (Desktop, Tablet, Mobile).
- Quản lý giỏ hàng với `localStorage`.
- Danh mục sản phẩm, bộ lọc tìm kiếm.
- Dữ liệu được fetch từ file JSON cục bộ (không cần server/backend).

## Hướng dẫn chạy dự án
1. Mở thư mục `wood-furniture-shop` trong VS Code.
2. Cài đặt extension **Live Server** (nếu chưa có).
3. Nhấp chuột phải vào `pages/index.html` và chọn **Open with Live Server**.
4. Trình duyệt sẽ tự động mở tại địa chỉ `http://127.0.0.1:5500/pages/index.html`.
