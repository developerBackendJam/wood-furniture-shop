# Yêu cầu dự án (Requirements)

## 1. Công nghệ
- HTML5, CSS3, JavaScript thuần.
- Không sử dụng thư viện (như Bootstrap, Tailwind, jQuery, React).

## 2. Dữ liệu
- Lưu trữ dữ liệu tĩnh bằng định dạng JSON (`database/products.json`, `database/categories.json`).
- Fetch dữ liệu bằng JavaScript Fetch API.

## 3. Chức năng chính
- **Trang chủ**: Hero banner, Sản phẩm nổi bật, Danh mục.
- **Trang sản phẩm**: Danh sách sản phẩm, Tìm kiếm, Lọc, Sắp xếp.
- **Chi tiết sản phẩm**: Hiển thị thông tin sản phẩm chi tiết.
- **Giỏ hàng**: Thêm, xóa, cập nhật số lượng, lưu bằng `localStorage`.
- **Thanh toán**: Biểu mẫu nhập thông tin giao hàng, hiển thị tóm tắt đơn hàng.
- **Liên hệ**: Hiển thị form và thông tin liên hệ.

## 4. Giao diện (UI/UX)
- Layout bằng Flexbox/CSS Grid.
- Giao diện đáp ứng (Responsive).
- Màu sắc chủ đạo: Đen (`#1F1F1F`) và màu gỗ tự nhiên.
